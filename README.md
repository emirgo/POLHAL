## POLHAL
#### Open source HAL for stm32f4xx
Design of this HAL is currently based on stm32f446RET6. If need be it can be extended further. <br><br>
Road map:<br>
<ul>
	<li>GPIO</li>
	<li>Print</li>
	<li>Timers</li>
	<li>PWM</li>
	<li>SysTick</li>
	<li>ADC</li>
	<li>UART</li>
	<li>DMA</li>
	<li>i2c</li>
	<li>SPI</li>
</ul>
